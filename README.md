# Jay Patel Portfolio

This repository contains a personal portfolio website for Jay Patel, a Java Backend Engineer specializing in Spring Boot, Apache Beam, GCP, cloud-native data pipelines, and secure backend architecture.

## Overview

- Static portfolio website built with HTML and CSS
- Showcases professional experience, skills, projects, certifications, and contact details
- Includes clickable certificate badges and clear section navigation

## Key Features

- Responsive home and about sections
- Skills section with modern card styling and descriptions
- Simplified project cards optimized for readability
- Credential links to public Credly badges
- Contact form integration via Web3Forms

## Technologies

- HTML
- CSS
- Font Awesome

## Files

- `index.html` — main portfolio page
- `assets/css/styles.css` — styling and layout
- `assets/img/` — profile and technology images

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/pjay7/Portfolio.git
   ```
2. Open `index.html` in a web browser.
